# Agent Skills Specification

The canonical Agent Skills specification is hosted at:

<https://agentskills.io/specification>

When working with this skill, fetch that URL whenever you need the
authoritative spec for SKILL.md frontmatter, file layout, progressive
disclosure rules, or compatibility constraints. The contents at that
URL supersede the writing-pattern guidance in `SKILL.md` if they ever
disagree.

The repo-level `spec/` directory in upstream `anthropics/skills` only
contains a pointer to this URL — the spec itself is intentionally
maintained out-of-tree.
